## Installation Instructions
1. Clone the repository:
   
   ```bash
   git clone https://github.com/khalid-arrazani/My-Portfolio.git
   ```

2. Navigate into the project directory:
   
   ```bash
   cd My-Portfolio
   ```

3. Install dependencies:
   
   ```bash
   npm install
   ```

4. Start the application:
   
   ```bash
   npm start
   ```
